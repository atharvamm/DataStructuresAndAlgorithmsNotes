Merge Two Sorted Lists

Approach

In this problem we are given two sorted linked lists and we are supposed to merge them. There are two approaches to do this.

In the first approach we extract all the nodes of each linked list to a list and then compare and add them to make a new list. We then use this new list to create a new linked list by changing the next element based on their index in the list.

A memory efficient way to do this would be to first create a dummy head and then just move the pointer for this new head by one step based on which ever node element has lesser value.
