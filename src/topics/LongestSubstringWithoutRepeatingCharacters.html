Longest Substring Without Repeating Characters

Approach

In this problem we need to find the substring with non-repeating characters. Since its a substring our goal is to ensure the elements in the array are consecutive.

Given that elements should be consecutive means we can each element at a time and save it in a set. If the element already is in the set, we can shorten the string until that element is no longer in the set.

Main Ideas

  • Use sets when you want to see if the element was seen before.
  • In problems when you have two pointers moving, first move the end pointer on the base test or the problem that you are trying to solve and if that fails start moving the start pointer until the sub-array/sub-sequence solves that problem. Do this until end reached the end of the array.
