Graph Valid Tree

Approach

What this problem asks from us is to confirm if the given graph is a valid tree. Since trees are essentially graphs without loops, the solution involves looking for loops.

This can be done by getting child nodes for each nodes using edge information. Once we have node and children info, all we have to do is just go to each node and start from there to see if we can come back to a node already seen.
