Maximum Product Subarray

Approach

We want to find the subarray with max product. It is important to remember that a subarray means continuous elements so we have to ensure that out method takes that into account.

The easiest method would be to first get all possible sub arrays and then check their product. But this ones time complexity would be of O(n^2)

A better method would be to accumulate product from either sides and dump or restart in the worst case scenario cause as long as the numbers are not 0 they are contributing to the product. And if there is an odd number of negatives, since we accumulate from both directions it is likely we would have accumulated even number of negatives from either side of every odd number and seen its product.

Main Ideas

  • Cases where we have to find some sort of accumulated values for subarrays. The two most common methods are 1. Accumulate and then drop when condition changes, 2. Accumulate from both sides and then only drop when accumulating a particular element is detrimental. When solving problems based on accumulation these 2 methods usually help.
