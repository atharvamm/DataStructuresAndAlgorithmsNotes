Valid Parentheses

Approach

The question wants us to check if all combinations of brackets are complete or not. The main challenge here is to ensure that we first check the inner brackets and then start looking for outer brackets.

An efficient way of solving this problem would therefore be to first ignore the outer brackets and try solving the inner brackets. (Unless there are consecutive open-close pairs. In that case we will not have nested brackets)

So we try to check from either left to right or right to left. If we find that there is an instant solution we remove the pair if not add it to a new list/stack to check later.
