Move Zeroes

Approach

Trivial approach, first go through array and find all non zero elements and store them for later use. Then go through array again and all elements one by one. Fill remaining spots with zero.

We can use two pointer to avoid storing an extra array. We can have slow pointer which only updates on non - zero element and fast pointer which updates to go through the array. Once fast pointer reaches end of list, we use slow pointer to fill remaining elements in list with 0.
