Rotate Array

Approach

Rotation is essentially removing from one end and adding from another. A simpe method of doing this would be to use pop, which returns/removes the last element from the array and then use insert to add it to the appropriate index.

An issue with the pop, insert method is. Everytime we add an element we waste time moving all elements by one place. Instead we can use a store array to store all elements that needed to be added to the beginning of the array in one place and focus on moving the elements. Once we move the elements we can use the store add the elements to the beginning of the array. This way we only move the elements once.

There is another approach, which is optimal but not super intuitive.
