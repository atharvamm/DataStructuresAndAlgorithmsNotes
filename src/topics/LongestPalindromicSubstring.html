Longest Palindromic Substring

Approach

We are given a string and we are supposed to find the longest palindromic substring.

One way to do that is to go at each element and check what is the longest possible palindrome from there. A catch here is that each index there are 2 possible type of arrays. One where only the element at that index is part of the palindrome, the other where the neighbouring element is also part of the palindrome.
